Detailed Description

This is the class that represents the units in the game. The units have name,healt points,attack damage and attackcooldown. They can attack each other back and forth, until one of them dies.

This makes the units hit each other.

This fight function takes two unit pointers. Unit1 attacks the unit2 first, then according to the attackspeed parameters, units hit each other until one of them dies. <param name="unit1" This is the unit which attacks first, and the so called "agressor". <param name="unit2" This is the unit which "defends".
